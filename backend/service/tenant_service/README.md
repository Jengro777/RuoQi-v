# 租户服务

租户自服务与服务模块（HTTP + RPC），承载租户自服务门户。

## 在架构中的位置

```
中台管理 platform_tenant_portal_api  →  管理侧（审核、分配）
租户管理 tenant_portal_api           →  自服务侧（订阅、成员、账单）
```

## 业务范围

- 租户订阅平台产品（WMS、TMS、Mall 等），一个租户可同时开通多个平台
- 邀请外部用户加入租户、账单与付款
- 宿主租户不在此管理（由中台运营方维护）

## 子门户

| 目录 | 用户 |
|---|---|
| `tenant_portal_api` | 企业租户管理员 |
