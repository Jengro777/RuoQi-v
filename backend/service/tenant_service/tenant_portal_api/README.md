# 租户自服务门户

企业租户管理侧入口，终端用户日常不接触。宿主租户不在此管理（宿主租户由中台运营方维护）。

## 业务边界

- 订阅平台产品（WMS、TMS、Mall 等），一个租户可同时开通多个平台
- 邀请外部用户加入租户（从用户池中选择，不新建账号）
- 账单与付款
- 不涉及终端用户的日常操作

## 账户模型

- 外部用户由平台统一注册，可通过宿主租户的默认工作区使用应用
- 外部用户被企业租户邀请后，可创建绑定企业租户的工作区
- 切换工作区即切换租户上下文，用户无需感知租户概念
- 中台账号不在此处管理（见 `platform_service/platform_portal_api`）

## 注意

与 `platform_service/platform_tenant_portal_api` 区分：这里是企业租户自己操作，中台管理员的租户管理在 `platform_service` 下。
